# CLI JSON Formatter Validator

This Python script provides a command-line utility to format and validate JSON data. It serves as an efficient alternative to using online JSON tools in a browser, helping developers reduce context switching and maintain focus. By processing JSON directly from the terminal, it aligns with the article's goal of minimizing reliance on multiple browser tabs for simple development tasks.

## Language

`python`

## How to Run

1. Save the code as `json_tool.py`.
2. Run with piped input: `echo '{"name": "Alice", "age": 30}' | python json_tool.py`
3. Run with a file: `python json_tool.py data.json` (where `data.json` contains valid JSON)
